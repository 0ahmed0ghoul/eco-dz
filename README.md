# eco-dz 🏞️

A user-friendly web application designed to showcase and promote natural monuments.

## Table of Contents 📜

- [Features](#features-)
- [Installation](#installation-)
- [Usage](#usage-)
- [Contributing](#contributing-)
- [License](#license-)

## Features 🌟

*   **Discover Natural Monuments:** Explore a curated list of breathtaking natural landmarks.
*   **User-Friendly Interface:** An intuitive and easy-to-navigate web application.
*   **Detailed Information:** Access comprehensive details about each monument.
*   **Interactive Map Integration:** Visualize monument locations on an interactive map.
*   **Image Galleries:** View stunning visual content for each natural site.
*   **User Reviews & Ratings:** Share your experiences and read reviews from other users.
*   **Search Functionality:** Quickly find specific monuments or types of natural attractions.

## Installation 🛠️

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have the following installed on your system:

*   [Node.js](https://nodejs.org/) (which includes npm)

### Cloning the Repository

Clone the eco-dz repository to your local machine:

```bash
git clone https://github.com/0ahmed0ghoul/eco-dz.git
cd eco-dz
```

### Frontend Setup

Navigate to the `frontend` directory and install the dependencies:

```bash
cd frontend
npm install
```

## Usage 🚀

Once the installation is complete, you can start the development server.

### Running the Application

From the `frontend` directory, run the following command:

```bash
npm run dev
```

This will start the development server, and you can access the application in your web browser, typically at `http://localhost:5173` (the port might vary).
